# Zorvyn Backend Assignment

## 📌 Project Description
This is a Spring Boot backend application for managing transactions.

## 🚀 Features
- Create Transaction
- Get All Transactions
- Update Transaction
- Delete Transaction
- Summary API

## 🛠️ Tech Stack
- Java
- Spring Boot
- MySQL
- Maven

## 🔗 API Endpoints

### GET
/transactions

### POST
/transactions

### PUT
/transactions/{id}

### DELETE
/transactions/{id}

### SUMMARY
/transactions/summary

## ▶️ How to Run
1. Import project in Eclipse
2. Run as Spring Boot App
3. Test APIs using Postman
